# Contributing to BOSC Community Library

## How Can I Contribute?

### Reporting Bugs
- Use the Bug Report template
- Include steps to reproduce

### Suggesting Features
- Use the Feature Request template
- Explain the problem and solution

### Submitting Pull Requests

1. Create a feature branch:
   - `git checkout -b feature/your-feature-name`
   - `git checkout -b fix/your-bug-fix`

2. Make your changes and commit:
   - `git commit -m "feat: add something - fixes #N"`

3. Push and open a Pull Request

## Branch Naming Convention

| Type | Branch name |
|------|-------------|
| Bug fix | `fix/brief-description` |
| Feature | `feature/brief-description` |
| Refactor | `refactor/brief-description` |

## Code of Conduct

By participating, you agree to follow our Code of Conduct.
